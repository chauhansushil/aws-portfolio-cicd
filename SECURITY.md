# Security Policy

## Supported Versions

This project is currently maintained and receives security updates.

| Version | Supported |
| ------- | ---------- |
| 1.0.x   | ✅ Yes     |
| < 1.0   | ❌ No      |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

Instead, report it responsibly by:

- Email: your-email@example.com
- Or create a private security advisory via GitHub

We will respond within 48 hours.

---

## Security Measures Implemented

This project follows cloud security best practices:

- IAM Least Privilege
- S3 Bucket Policy Restrictions
- CloudFront Origin Access Control (OAC)
- Server-Side Encryption Enabled
- CI/CD Security Scanning (CodeQL, npm audit)

---

## Disclosure Policy

We follow responsible disclosure guidelines. Vulnerabilities will be patched and publicly disclosed after resolution.
